商务智能

2023/五月/15 11:11 发布在 IT

第一章

1、商务智能产生的原因

  1. 商务智能服务企业战略：商务智能可以对企业的内、外部数据进行分析，支持企业战略管理
  2. 商务智能提升企业绩效：通过商务智能可以从企业多年运营的数据中，挖掘有效的模式辅助管理决策
  3. 商务智能提炼知识辅助决策：商务智能对业务系统中生成的运营数据进行分析，给出报告，帮助管理者认识企业和市场现状，预测发展趋势，做出正确决策
  4. 商务智能是多项技术的综合应用：商务智能应用了多种技术，帮助企业发现数据后隐藏的商机或威胁，进而推动企业业务发展
  5. 商务智能用户多样：商务智能服务于企业的各种用户使用

2、商业智能对企业的价值

  1. 制定合适的市场营销策略：利用商务智能技术构建商业模型，确定合适的营销策略。把业务系统的数据整合到数据仓库后通过挖掘得到各类消费者的消费习惯，从而精确的投放广告和促销计划
  2. 改善顾客管理：顾客智能是商务智能在顾客关系管理中的应用。商务智能使用OLAP和DM等技术，处理交易记录和相关顾客资料，将顾客分类后制定相应的服务策略
  3. 降低经营成本提高收入：应用商务智能企业的绩效管理功能，可以低成本的指定各种收益报表，对不同的业务进行核算，深入分析偏差和改进方法，进而降低成本，提高收入
  4. 提高风险管理能力：商务智能可以识别潜在的危险，给出存在欺诈行为的用户特征。
  5. 改善业务能力：商务智能减少管理者收集数据，获取信息花费的时间，加速决策过程，使正确的信息在争取的时间流向决策者
  6. 提高市场响应能力：借助商业智能可以预测市场变化，精简流程，确定需要改进的环节，以适应外部环境的变动

3、商务智能系统和传统决策支持系统有什么关系

  1. 数据集成与分析能力：商务智能系统可以从多个数据源中提取、整合和加工数据，进行多维度的数据分析，帮助用户更好地理解数据的关系和趋势。传统决策支持系统也有数据分析的功能，但商务智能系统在数据集成和多维分析方面更为强大和全面。
  2. 数据可视化和报表功能：商务智能系统提供丰富的数据可视化方式，如图表、仪表盘、交互式报表等，以直观、易懂的方式展示数据和分析结果。传统决策支持系统也提供报表和图表等展示功能，但商务智能系统的可视化能力更加强大和灵活。
  3. 实时性和即席查询：商务智能系统通常能够提供实时的数据更新和即席查询功能，使决策者能够获取最新的数据和信息。传统决策支持系统可能更偏向批处理或周期性的数据更新和分析。
  4. 用户自助性和交互性：商务智能系统注重用户的自助性和交互性，使用户能够根据自己的需求进行数据查询、分析和报表设计。传统决策支持系统可能更多地由技术人员设计和维护，用户参与度较低。

第二章

1、商务智能系统包括哪些部分？分别有什么功能？

  1. 数据源与数据提取：这一部分负责从各种数据源中收集和整合数据，包括企业内部的数据库、数据仓库、操作系统日志等。业务数据需要经过数据评价、数据筛选以及ETL（抽取、转换、装载）后存储在数据仓库中，为数据的分析奠定基础
  2. 数据仓库：数据仓库的数据包括元数据和经过ETL的业务数据。数据源中的数据按照元数据库的规则，经过抽取、清理、转换、集成，按照决策主题重新组织，存储
  3. 访问工具：访问工具包括应用接口和中间件服务器。数据库中间件允许用户透明的访问数据仓库服务器，用于即席查询，在线分析处理和数据挖掘
  4. 决策支持工具：用于辅助决策制定和问题解决的软件或应用程序。这些工具提供了各种功能和技术，帮助决策者收集、分析、评估和可视化数据，从而做出更明智的决策
  5. 商务智能应用：从数据仓库中提取相关的数据，然后确定数据分析的方法，并把分析结果通过前端工具展示给决策者
  6. 系统管理：包括系统安全管理、元数据的管理与更新、数据仓库的日常维护与监控、数据使用审计和容量规划等
  7. 元数据管理：元数据管理包括对开发、管理数据仓库时所用的技术元数据和支持业务人员的业务元数据进行管理，他对数据仓库的设计和维护起着重要作用

第六章

1、什么是移动商务智能？移动商务智能在哪些领域得到了应用？

移动商务智能是指将商务智能技术和应用移植到移动设备上的一种信息技术解决方案。商务智能是指通过收集、分析和解释企业内外部的数据，为决策者提供支持和指导的一种技术和方法。移动商务智能则是将这种商务智能的能力扩展到移动平台上，使用户能够随时随地获取和分析相关的业务数据。

应用领域：

  1. 移动信息服务：实时信息服务。例如天气，新闻，股市信息等
  2. 移动广告：移动终端易于定位，便于搜集用户个人信息，因此移动终端上的广告更具有针对性
  3. 移动销售：用户可以随时随地购物。打破传统购物限制，收到消费者欢迎
  4. 移动办公：使许多员工可以经常外出，在途中收发邮件、文件、短信等。移动办公提高了移动工作者的效率
